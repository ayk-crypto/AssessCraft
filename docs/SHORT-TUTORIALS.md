# Short tutorial scripts

## Create an assessment in three minutes

1. Open AssessCraft → Templates.
2. Preview a starter and choose **Use this template**.
3. Rename the assessment and review Overview.
4. Edit stages and questions in Builder.
5. Confirm score bands and profiles.
6. Publish and copy the shortcode.

## Publish with Elementor

1. Publish the assessment.
2. Edit the destination page with Elementor.
3. Add the AssessCraft widget.
4. Select the assessment ID and update the page.
5. Test the complete response flow while logged out.

## Enable privacy-first lead storage

1. Open Lead Form and enable consultation requests.
2. Enable WordPress storage only if needed.
3. Write a specific consent label.
4. Update the assessment.
5. Set a retention period under Consultation Requests.
6. Submit a test request and verify export/erasure.

## Upgrade safely

1. Back up the database and files.
2. Clone production to staging.
3. Install the update on staging.
4. Review migration logs and existing assessments.
5. Test reports, leads, embeds, and integrations.
6. Schedule production deployment with rollback access.
